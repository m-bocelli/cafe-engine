# Cafe Engine
Minimalist 2D rogue-like engine built on SDL2.

## Compiling
To compile a debug executable on MacOS run the following on your command line:

```
make debug_m
```

For a release executable (optimized and w/o debug flags) run:

```
make release_m
```
